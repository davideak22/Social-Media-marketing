# Tech Stack & Tools

- **Core:** React 18 + Vite (Fastest local dev server)
- **Language:** TypeScript (Strict mode to prevent runtime crashes)
- **Styling:** Tailwind CSS (v3.4+)
- **Animation:** `framer-motion` (The primary engine for transitions)
- **State:** `zustand` (For managing `currentSceneIndex`)
- **Icons:** `lucide-react`
- **Utils:** `clsx`, `tailwind-merge`
